# MD - HTML - PDF
A project of playing with ways to make Markdown documentation readable and shareable everywhere.


```bash
npm start -- <INPUT_MARKDOWN_FILE> <OUTPUT_HTML_FILE>
```
OR
```bash
node index.js <INPUT_MARKDOWN_FILE> <OUTPUT_HTML_FILE>
```
